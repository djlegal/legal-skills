# 课程生成器

## 概述

从多个文献内容中系统化提取并生成结构化课程。支持 1-100+ 个文献文件，自动进行主题识别、课程大纲生成、章节内容撰写。

**核心原则**：
- **忠实原文**：保留文献中的核心观点、具体例子、重要表述，不添加文档外信息
- **段落式写作**：避免要点罗列，包含完整的论证和阐述
- **详尽完整**：每个章节包含核心观点、案例分析、重要表述、数据支撑、背景信息

## 工作流程

```
用户请求 → 读取文献 → 生成课程大纲 → 逐章节生成 → 保存文件
```

### 输入类型

1. **文献目录**：包含多个文献文件的目录（支持 md、txt 等格式）
2. **文献文件**：1-100+ 个文献文件
3. **输出位置**：与输入目录同级，自动创建以课程名称命名的文件夹

### 输出结构

```
课程名称/
├── 01_第一章主题.md
├── 02_第二章主题.md
├── 03_第三章主题.md
└── [更多章节文件]
```

## 执行步骤

### Step 1: 读取文献文件

使用 `Glob` 工具一次性读取指定目录下的所有文献文件：

```
Glob: {输入目录}/**/*.{md,txt}
```

保持完整上下文，一次性读取所有文件内容。

### Step 2: 生成课程大纲

调用 AI 模型，一次性分析所有文献并生成课程大纲。详见 [outline_prompt.md](references/outline_prompt.md)。

核心要求：
- **整体把握**：理解所有内容的整体脉络和逻辑关系
- **主题识别**：基于所有内容自动识别主要主题
- **忠实原文**：保留核心观点和表达方式
- **课程组织**：按主题生成章节，体现原文的逻辑结构

输出格式：
```
课程标题：[基于原文生成的课程名称]

第一章 [主题名称]
  1.1 [子主题] - 原文核心观点摘要
  1.2 [子主题] - 原文核心观点摘要

第二章 [主题名称]
  2.1 [子主题] - 原文核心观点摘要
  2.2 [子主题] - 原文核心观点摘要
```

### Step 3: 逐章节生成内容

为每个章节生成详细的段落式内容。详见 [chapter_prompt.md](references/chapter_prompt.md)。

章节内容结构：
1. **核心观点深度阐述** - 以段落形式详细阐述核心观点
2. **具体案例和例证分析** - 引用原文中的具体案例
3. **重要观点和表述** - 直接呈现重要观点和表述
4. **数据与事实支撑** - 数据和事实信息
5. **背景信息和延伸说明** - 背景信息和补充说明

### Step 4: 创建输出目录

- 使用 `Bash` 创建目录
- 输出路径：与输入目录同级
- 目录名：AI 生成的课程名称

### Step 5: 保存章节文件

为每个章节创建独立文件：`XX_[章节序号]_[章节标题].md`

## 质量标准

### 内容质量

- **忠实原文**：不改变原文含义，保留关键信息
- **言之有物**：段落式写作，避免要点罗列
- **详尽完整**：每个观点都有完整阐述
- **自然表达**：避免频繁使用"原文中"、"根据原文"等表述

### 格式规范

```
# 第X章 [章节名称]

## X.X [小节名称]

### 核心观点深度阐述
[段落式内容]

### 具体案例和例证分析
**案例一：**
[详细案例分析]

### 重要观点和表述
> "重要观点表述"

### 数据与事实支撑
- 数据一：[具体内容]

### 背景信息和延伸说明
[详细引用和分析]

## 章节核心内容汇总
- 核心观点汇总：[基于文献的要点梳理]
```
