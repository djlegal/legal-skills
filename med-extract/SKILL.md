# Med Extract - 医疗纠纷病历要点抽取

## 概述
- 面向医疗纠纷律师的病历信息抽取技能，优先完成“就诊人信息 + 诊疗时间顺序 + 每个时间点的具体事件”三件事。
- 支持处理中文病历首页、病程记录、手术知情同意书、手术记录、护理记录、出院小结等非结构化文本；如输入为扫描件或图片，需先获得 OCR 文本。
- 抽取模板与输出提示详见 `references/extraction_schema.md`（字段模板），使用时按模板输出，缺失项标注“未提及/待补充”，避免猜测。

## 适用场景
- 用户为医疗纠纷律师，希望快速了解病人诊疗经过、手术情况、并发症及高风险因素。
- 输入包含多份病历/病程/知情同意/手术记录，需要汇总为可检索、可比对的结构化信息。
- 需要初步提示潜在的医疗风险/过错线索（如诊疗延误、知情同意缺口、感染控制不足、慢病管理缺失）。

## 输入要求与预处理
- 接受 Markdown/纯文本；若为 PDF/图片，需说明是否已 OCR，必要时先提取文本再执行抽取。
- 先粗分文档类型（病历首页、入院记录、病程记录、手术记录、护理记录、检验报告、出院记录/小结、费用清单）。
- 处理常见噪音：去除重复水印、表格标签占位、乱序段落；保留时间戳、术式名称、检验指标与数值。

## 工作流程

### 1) 收集与拆分
- 聚合所有提供的文本，按时间与文档类型粗排顺序；保留原始片段以便引用。
- 对照 `extraction_schema.md` 的字段模板，确认可提取内容和缺失项。
- 注意：许多文书（医患沟通、同意书、检验报告等）在文件夹中可能集中放置，需依据签署/报告时间重新插入时间轴，而非按文件顺序。

### 2) 抽取要素（按模板）
- 提取就诊人信息：姓名、性别、年龄、身份证/病历号、婚姻/职业、联系方式。
- 提取就诊信息：医院/科室、床位号、入院途径、入院/出院时间、住院天数、医保/自费方式。
- 逐时间点记录“发生了什么”，每条至少包含：
  - 当时诊断/判断（入院诊断、阶段性评估、术前判断）。
  - 手术/操作（术式、术者、麻醉方式、术中所见）。
  - 术后监测与处置（生命体征、引流/负压、用药、并发症观察）。
  - 风险告知/知情同意（手术/麻醉/特殊治疗等的风险、替代方案、签署时间）。
  - 沟通记录（医患/家属沟通）按签署/记录时间放入对应节点（术前、诊断更新、麻醉前后等），即便原文件放在后部。
- 抽取关键检验/影像：标注日期、指标数值与异常方向（↑/↓）。

### 3) 质量校验
- 校对时间先后关系（入院→手术→复查→出院），标记矛盾或缺失（如出院时间缺、诊断不一致、手术知情同意缺失）。
- 核查慢病管理与感染控制是否贯穿（如糖尿病控制记录、抗菌药物使用、引流/换药频次）。
- 对无法确认的信息标注“未提及”而非推测。

### 4) 输出格式（推荐：分多文档交付）
一次执行产出多份 Markdown，以便律师快速落地：
- `病历概要.md`：结构化病历概要 + 诊疗时间轴（就诊人信息、时间节点详述、风险/告知要点）。
- `过错初步分析.md`：基于时间轴的初步过错/风险提示（诊疗延误、知情同意缺口、感染控制、慢病管理、监测护理不足等），引用对应时间节点。
- `律师行动清单.md`：律师后续行动清单（需补的材料/影像/检验、需重点询问的医护人员、可能的鉴定要点、人损评定准备、证据保全与沟通策略）。

示例（摘自 `record-summary.md`）：

```
就诊人信息：
- 患者：张三，男，34 岁；身份证号 000000000000000000；婚姻已婚。
- 医院/科室：某市立医院运动医学骨科；入院 2025-08-11 14:27，出院 2025-09-02 10:15，住院 22 天；入院途径：门诊。

诊疗时间轴（每条记录都写清“做了什么”）：
- 2025-07-17
  - 诊断/判断：软组织超声提示肌肉撕裂伴血肿。
  - 手术/操作：无。
  - 术前判断：未提及。
  - 术后监测/处置：未提及。
  - 风险告知/同意：未提及。
  - 证据：B 超检查记录。
- 2025-08-14
  - 诊断/判断：下肢感染需清创。
  - 手术/操作：皮肤和皮下组织非切除性清创 + VSD 负压；全麻；术中见大量脓液与坏死组织。
  - 术前判断：拟行清创 + VSD。
  - 术后监测/处置：负压覆盖创面，持续引流。
  - 风险告知/同意：手术风险告知书与知情同意签署情况需核查。
  - 证据：手术记录/知情同意书。

风险与告知要点：
- 手术风险告知是否完整（含并发症、替代方案、签署时间）。
- 术后监测是否连续记录（引流、体温、血象、感染控制）。
```

### 5) 结果检查与交付
- 确认已覆盖字段表的关键要素，未提及项明确标注。
- 输出引用/摘录时附来源（例如“源自：2025-08-11 入院记录”），方便律师回查。
- 如发现明显矛盾或缺页，单列“待补充材料”列表。

## 过错分析提示（使用时仅作线索，不作定性）
- 诊断/治疗延误：症状持续时间长、入院前多次就诊未处理、入院后关键检查/用药延迟。
- 感染控制：反复清创/负压仍感染，抗菌药物方案不清、菌培养或药敏缺失、引流不畅/换药不足。
- 慢病与并存疾病管理：糖尿病血糖控制记录缺失、抗结核未启动、贫血/血小板异常未处理。
- 手术与知情同意：多次手术/清创的适应症与术后评估记录缺失，知情同意与手术安全核查是否完整。
- 记录完整性：出入量、生命体征、护理记录缺口；费用异常与医疗行为不匹配。
