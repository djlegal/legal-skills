# 病历信息抽取模板与提示

本文件不是“强制 schema”，而是一个**字段模板**：用于统一抽取口径，确保不同病历的输出一致、可对比。最终产出是结构化的 Markdown 文档。

## 核心字段模板

### 患者与就诊信息
- `patient`: 姓名、性别、年龄/出生日期、身份证号/病历号、婚姻、职业、联系方式。
- `encounter`: 医院与科室、床位号、入院途径（门诊/急诊/转院）、入院时间、出院时间、实际住院天数、医保/自费方式。

### 诊断与风险
- `chief_complaint` 与 `present_illness`: 主诉、病程长短、受伤/感染起因。
- `diagnosis`: 入院诊断、出院主要诊断、其他诊断/并存疾病（慢病、感染、外伤）、损伤外因、过敏史。
- `risk_factors`: 糖尿病控制情况、免疫抑制/结核/肥胖/吸烟饮酒史等。

### 手术与操作
- 记录列表，每项包含：日期时间、术式名称、手术/操作级别、术者/助手、麻醉方式与 ASA 分级、适应证、术中所见、关键操作（清创/缝合/VSD/植入物）、出血量/输血、术后引流与处理、术后并发症/恢复情况。
- 常见关键词：`拟行`、`手术日期`、`切口`、`坏死组织`、`VSD`、`负压`、`清创`、`麻醉方式`、`ASA`、`术中所见`。

### 治疗与护理
- 药物：抗菌药物、镇痛、抗凝/预防血栓、降糖/胰岛素、抗结核等，记录药名/途径/频次如可见。
- 伤口/引流/换药：VSD 负压时间、换药频次、湿敷、抬高患肢等。
- 支持治疗：输血/血制品、补液、氧疗、康复训练要点。

### 检查与检验
- 影像：B 超/CT/MRI/DR 所见与结论。
- 实验室：WBC、CRP/超敏 CRP、降钙素原、血红蛋白、血小板、血糖、肝肾功能、电解质、菌培养/药敏。
- 取时间点与异常值，保留单位；可用“日期+项目+数值+参考/异常”格式。

### 住院时间轴
- 以日期时间为序列，列出关键事件与来源：入院、主要检查、手术/操作、病程转折、并发症、重要检验结果、出院。
- 每个时间点至少包含以下要素（缺失标注“未提及”）：
  - `diagnosis_at_time`: 当时诊断/判断（含术前判断）。
  - `procedure_at_time`: 手术/操作（术式、术者、麻醉、术中所见）。
  - `postop_monitoring`: 术后监测与处置（生命体征、引流/负压、用药、并发症观察）。
  - `consent_and_risk`: 风险告知/知情同意（是否签署、告知内容、替代方案）。

### 费用与结局（可选）
- 费用概要：总费用与主要构成（治疗/手术/耗材/药物）。
- 结局：出院方式（医嘱离院/转院/其他）、随访/复诊建议。

### 不良事件/并发症
- 感染未控、再手术、贫血/失血、低血糖/高血糖、血栓/压疮、引流不畅等。

## 抽取与整理提示
- 优先引用带时间戳的记录；同一信息多次出现时，保留最清晰/最新的版本，其他版本可在时间轴中呈现。
- 使用 `YYYY-MM-DD HH:MM` 标注时间（无时间则仅写日期）；缺失值写“未提及/待补充”，避免编造。
- 诊断与术式尽量使用原文表述；如存在 ICD 编码可一并列出。
- 关键检验值同时标注异常方向（↑/↓）；如参考范围缺失，可根据描述（升高/降低）判断方向。
- 时间轴每条建议包含：`时间 | 事件 | 证据摘录 (来源文件/段落)`。

## 示例输出片段（Markdown，匿名示例）

```markdown
就诊人信息：
- 患者：李四，男，34 岁；身份证号 000000000000000000。
- 医院/科室：某市立医院运动医学骨科；入院 2025-08-11 14:27，出院 2025-09-02 10:15；住院 22 天。

诊疗时间轴：
- 2025-07-17
  - 诊断/判断：软组织 B 超提示跖肌撕裂伴血肿 35×33×8mm。
  - 手术/操作：无。
  - 术前判断：未提及。
  - 术后监测/处置：未提及。
  - 风险告知/同意：未提及。
  - 证据：B 超记录。
- 2025-08-14
  - 诊断/判断：右下肢感染需清创。
  - 手术/操作：皮肤和皮下组织非切除性清创 + VSD 负压；全麻；术中见大量脓液与坏死组织。
  - 术前判断：拟行清创 + VSD。
  - 术后监测/处置：负压覆盖创面，持续引流。
  - 风险告知/同意：手术风险告知书与知情同意签署情况需核查。
  - 证据：手术记录/知情同意书。

风险与告知要点：
- 知情同意是否完整（并发症、替代方案、签署时间）。
- 术后监测记录是否连续（引流、体温、血象、感染控制）。
```
